#Açıklama
Verilen URL'deki bütün linkleri çekiyor. Çekilen url'leri path (/example) olarak ayırıp tekrar bir link bulma işlemi yapıyor.

#Kullanımı
python dereliUrl.py -u https://www.example.com -o example.txt
